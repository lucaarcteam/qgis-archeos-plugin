-------------------------------------------
WHAT IS QGIS-ARCHEOS-PLUGIN?
-------------------------------------------

This package was built by Alessandro Bezzi and Luca Bezzi to implement ArcheOS version of QGIS with some extra plugins (check the plugins for their specific license).
The package is released under the GNU General Public License and it is intended to be a transitional (temporary) package to ArcheOS 5 (Theodoric).
